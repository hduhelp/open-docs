# 杭电助手开发文档

本仓库是杭电助手文档项目，基于[vuepress2](https://v2.vuepress.vuejs.org/)构建。

### 2、本地克隆代码开发

```sh
git clone git@github.com:hduhelp/open-docs.git
cd bot-docs
pnpm install
pnpm docs:dev
```
更多命令请见 [contributing.md](./contributing.md)
